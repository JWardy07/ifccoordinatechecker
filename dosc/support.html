IFC Coordinate Checker Support

Support page for IFC Coordinate Checker by Jamie Ward.

About the app

IFC Coordinate Checker helps BIM and digital construction teams check whether IFC models are correctly positioned, georeferenced and ready for coordination. The app reviews IFC file information, units, georeferencing signals, TrueNorth, storeys, model extents and coordinate risk.

Common uses

  • Check whether an IFC model is far from origin.
  • Review whether georeferencing data is present.
  • Check units and coordinate extents before federation.
  • Export a CSV report for coordination records.
  • Use the built-in assistant to explain deterministic coordinate findings.

Supported files

IFC Coordinate Checker V1 supports plain .ifc files. IFCZIP support is planned for a future update.

Privacy

IFC files are selected by the user and processed locally on-device. The app does not require cloud upload to check a model.

Need help?

For support, bug reports or feedback, please email: jamiepward@googlemail.com

Useful information to include

  • Your device type: iPhone, iPad or Mac.
  • Your iOS, iPadOS or macOS version.
  • The IFC authoring tool, if known, such as Revit, Tekla, Vectorworks, Civil 3D or Solibri.
  • A short description of the issue.
  • Whether the file is a plain .ifc file or another format.
